# WishlistBot 🎁

WishlistBot — это бот для управления списками желаний.  
Он помогает пользователям создавать, хранить и управлять вишлистами, добавлять и удалять товары, а также просматривать список желаний через команды бота.

## 🚀 Возможности

* 📌 Создание персональных вишлистов
* ➕ Добавление товаров в список желаний
* ❌ Удаление товаров из списка
* 📋 Просмотр текущего вишлиста
* 🔍 Поиск и получение информации о товарах (если реализовано)
* 💾 Хранение данных пользователей

## 🧠 Идея проекта

Проект создан для упрощения процесса управления списками желаний.  
Вместо заметок и разрозненных ссылок пользователь может централизованно хранить свои желания в одном месте через бота.

## 🛠 Технологии

* Python
* Telegram Bot API
* SQLite
